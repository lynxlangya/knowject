# docs/

`docs/` is the only project documentation root and source of truth for Knowject.

## Truth Surfaces

| Surface | Directory |
| --- | --- |
| Current facts | `docs/current/` |
| Contracts | `docs/contracts/` |
| Standards | `docs/standards/` |
| Plans | `docs/plans/` |
| Handoff | `docs/handoff/` |
| Roadmap | `docs/roadmap/` |
| Inputs | `docs/inputs/` |
| Design | `docs/design/` |
| Templates | `docs/templates/` |
| Derived exports (not truth surface) | `docs/exports/` |

## Default Reading Order

- Codex: `docs/current/` -> `docs/contracts/` -> `docs/standards/` -> task-specific folders.
- Human collaborators: `docs/current/` -> `docs/roadmap/` -> `docs/plans/` -> task-specific folders.

`docs/superpowers/*` is process-asset space for workflows and execution support, not a project truth surface.
