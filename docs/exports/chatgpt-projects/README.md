# ChatGPT Projects Exports

`docs/exports/chatgpt-projects/*` 是派生导出层，不承担事实源职责。导出前始终先更新 live docs，再同步这里。

当前建议一并导出的主入口：

- `AGENTS.md`
- `docs/current/architecture.md`
- `docs/current/project-chat-sources.md`
- `docs/contracts/auth-contract.md`
- `docs/contracts/chat-contract.md`
- `docs/contracts/chroma-decision.md`

如果只需要最小项目包，可优先使用：

- `PROJECT_RULES.md`
- `CURRENT_ARCHITECTURE.md`
- `PROJECT_BRIEF.md`
- `GAP_ANALYSIS.md`

如果要把 Knowject 作为简历核心项目展示，可额外使用：

- `RESUME_CORE_PROJECT.md`
